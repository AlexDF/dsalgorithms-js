    Function maxArray

    Compares two values and returns the larger one.

    • maxArray<T>(array: T[]): T

      Finds the maximum value in an array of comparable elements.

      Type Parameters

      • T

        The type of elements in the array (must be comparable).

      Parameters

      • array: T[]

        The array to search for the maximum value.

      Returns T

      The maximum value found in the array.

      Throws

      When the array is empty or contains uncomparable elements.

      Example

      // Array of numbers
      const maxNum = maxArray([1, 6, 2.25, 7, 4]);
      // Returns: 7

      Example

      // Array of strings
      const maxString = maxArray(['banana', 'orange', 'apple']);
      // Returns: 'orange'
